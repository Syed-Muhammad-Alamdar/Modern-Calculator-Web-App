# Modern Calculator Web Application

A modern multi-page calculator web application built using **HTML**, **CSS**, **JavaScript**, and **LocalStorage**.

This project started as a simple calculator practice project and gradually evolved into a complete frontend web application featuring user authentication, password recovery, personalized greetings, and calculation history.

---

## Features

* User Registration (Sign Up)
* User Login
* Protected Calculator Page
* Password Recovery using Registered Phone Number
* Personalized Welcome Message
* Basic Calculator Operations
* Calculation History
* Date & Time Stored with Each Calculation
* Clear History Functionality
* Modern Glassmorphism Inspired UI
* About Page

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap (Basic Responsive Layout)
* LocalStorage API
* Google Fonts (Poppins)

---

## Project Structure

```
signup.html      → User registration page

log.html         → User login page

calc.html        → Main calculator

history.html     → Displays saved calculations

forget.html      → Password recovery page

about.html       → Information about the project

sign.js          → Handles registration

main.js          → Handles login validation

calc.js          → Calculator logic and history

his.js           → Displays calculation history

forget.js        → Password recovery logic
```
