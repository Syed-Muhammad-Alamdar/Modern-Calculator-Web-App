About This Project

This Calculator Web Application was developed as a frontend learning project using HTML, CSS, JavaScript, and LocalStorage. What started as a simple calculator gradually evolved into a complete multi-page web application with authentication, password recovery, personalized greetings, and calculation history tracking.

Features

  • User Login System
  • User Registration System
  • Password Recovery Using Phone Number
  • Calculator Operations
  • Calculation History with Timestamps
  • Personalized User Greeting
  • LocalStorage Data Persistence
  • Modern Glassmorphism-Inspired UI

Technologies Used

  • HTML5
  • CSS3
  • JavaScript (Vanilla JS)
  • Bootstrap
  • LocalStorage API

Purpose

The goal of this project was to strengthen my understanding of JavaScript, DOM manipulation, LocalStorage, multi-page application development, authentication concepts, and modern UI design techniques.
